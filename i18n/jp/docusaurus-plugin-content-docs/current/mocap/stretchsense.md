# StretchSense Glove

[StretchSense Glove](https://stretchsense.com/)による指のトラッキング。

## 設定
Warudoは、[VMCプロトコル](./vmc)を介して StretchSense Gloveをサポートします。StretchSense GloveをWarudoで使用するには、初心者ナビの基本設定中に、**StretchSense Glove** を **[Secondary Pose Tracking]** として選択します。

![](/doc-img/jp-stretchsense-1.png)

## キャリブレーション

StretchSense Gloveのキャリブレーションは、StretchSense Hand Engineで行います。
